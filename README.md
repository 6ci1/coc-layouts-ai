# COC阵型工具 - AI编程

部落冲突（Clash of Clans）阵型浏览与跳转工具。

**本项目完全由 AI 辅助编程完成。** 从零到上架，所有代码编写、架构设计、调试优化均由 AI 与人类开发者协作实现。这是一个 AI 编程能力的实践案例。

## 功能

- 从腾讯文档自动拉取阵型图片和链接
- 图片网格展示，点击跳转游戏应用阵型
- 支持国服/国际服筛选、TH等级筛选、排序
- 收藏功能，本地收藏管理
- 下拉刷新，智能缓存（数据 hash 比对，无变化不重新解析）
- 图片磁盘缓存，打开后本地加载，无需重复下载
- 自定义文档链接，支持手动粘贴阵型链接
- 历史记录，快速打开使用过的阵型
- 卡密验证系统（llua.cn V2 协议）

## AI 编程说明

本项目的开发过程体现了 AI 辅助编程的优势：

- **快速迭代**：从初始版本到功能完善，AI 能快速响应需求变更
- **架构设计**：MVVM + Repository 模式，Hilt 依赖注入，Room 本地缓存
- **代码质量**：遵循 Android 开发最佳实践，Compose 声明式 UI
- **问题排查**：AI 辅助定位编译、运行时问题并快速修复
- **安全意识**：敏感密钥与源码分离，通过 BuildConfig 注入

## 技术栈

- Kotlin + Jetpack Compose
- Material 3
- Hilt (依赖注入)
- Room (本地数据库)
- Retrofit + OkHttp (网络请求)
- Coil (图片加载与缓存)

## 构建步骤

1. 复制 `local.properties.template` 为 `local.properties`
2. 填入验证密钥（从 llua.cn 控制台获取）
3. 用 Android Studio 打开项目
4. 等待 Gradle 同步完成
5. 连接手机或启动模拟器，点击 Run

## 数据源配置

在应用内连击标题 6 次打开设置，可修改国服/国际服文档 ID。

文档ID就是腾讯文档URL中 `/doc/` 后面的那串字符。
